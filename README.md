# 🤞 Jujutsu Kaisen: Domain Expansion

A Python-based simulation of the most powerful Domain Expansions from the JJK universe. Experience the overwhelming power of the Limitless and the Divine Ferocity of the King of Curses.

## 🌌 Featured Domains

### 1. Unlimited Void (無量空処, Muryōkūsho)
* **User:** Satoru Gojo
* **Effect:** Grants the target infinite information, paralyzing them completely.

### 2. Malevolent Shrine (伏魔御厨子, Fukuma Mizushi)
* **User:** Ryomen Sukuna
* **Effect:** A barrierless domain that relentlessly slashes everything within its effective radius using Cleave and Dismantle.

---

## 🚀 Getting Started

### Prerequisites
Ensure you have **Python 3.11** installed on your system.

### Installation & Execution
To run the domain simulation, follow these steps to activate the specific virtual environment and execute the script:

1.  **Open your terminal/command prompt.**
2.  **Activate the environment:**
    ```bash
    # Windows
    venv311\\Scripts\\activate
    ```
3.  **Run the application:**
    ```bash
    python main.py
    ```

---

## 🛠️ Technical Details
- **Language:** Python 3.11
- **Environment:** Pre-configured `venv311`
- **Entry Point:** `main.py`
